ABOUT INFORMATION OF THIS PROJECT 
Library Management System
Developed a Java-based Library Management System to manage books and authors through a menu-driven interface.
Implemented CRUD operations for book and author management, including add, update, delete, and view functionalities.
Integrated MySQL database connectivity using JDBC for efficient storage and retrieval of library records.
Applied object-oriented programming principles and layered architecture (DAO, Service, and Model) to build a scalable and maintainable application.

Skills Used: Java, JDBC, MySQL, OOP, Database Management, CRUD Operations.
